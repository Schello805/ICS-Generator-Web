1. Datenschutz auf einen Blick

Allgemeine Hinweise

Diese Datenschutzerklärung klärt Sie über die Art, den Umfang und Zweck der Verarbeitung von personenbezogenen Daten innerhalb unseres ICS Generators und Validators auf.

Datenverarbeitung im Browser

Der ICS Generator und Validator verarbeiten alle Daten ausschließlich lokal in Ihrem Browser. Es werden keine Daten an Server übertragen oder extern gespeichert.

Cookies und Tracking

Diese Website verwendet keine Cookies. Zur statistischen Auswertung der Nutzung wird Matomo eingesetzt. Matomo läuft auf einem eigenen Server des Betreibers (Michael Schellenberger, 91572 Bechhofen), die Daten werden nicht an Dritte weitergegeben und bleiben ausschließlich beim Betreiber. Die Erfassung erfolgt anonymisiert und ohne die Verwendung von Cookies.

Externe Ressourcen

Diese Website lädt folgende externe Ressourcen:

  • Bootstrap CSS und JavaScript von stackpath.bootstrapcdn.com und cdn.jsdelivr.net
  • Font Awesome Icons von cdnjs.cloudflare.com
  • jQuery von code.jquery.com

Beim Laden dieser Ressourcen wird technisch bedingt Ihre IP-Adresse an die jeweiligen Anbieter (CDNs) übermittelt. Darüber hinaus werden keine personenbezogenen Daten an diese Dienste übermittelt oder von ihnen gespeichert.

2. Art der verarbeiteten Daten

  • Termindaten (Titel, Datum, Uhrzeit)
  • Beschreibungen
  • Ortsangaben
  • URLs und Anhänge
  • Erinnerungseinstellungen
  • Wiederholungseinstellungen

3. Zweck der Verarbeitung

Die Verarbeitung der Daten erfolgt zum Zweck der:

  • Erstellung von ICS-Kalenderdateien
  • Validierung von ICS-Dateien
  • Temporäre Anzeige von Validierungsergebnissen

4. Ihre Rechte

Da keine personenbezogenen Daten gespeichert werden, entfallen die üblichen datenschutzrechtlichen Auskunfts- und Löschungsrechte. Alle eingegebenen Daten werden nur temporär in Ihrem Browser verarbeitet und mit dem Schließen der Seite gelöscht.
